Ce dossier contient une archive des modules SQLAlchemy originaux.
Ils sont conservés à des fins de référence mais ne sont plus actifs dans le
déploiement Firestore. Si vous souhaitez restaurer la base SQL, copiez les
fichiers nécessaires hors de ce dossier et réactivez `database.py` pour
pointer vers une vraie base.

Fichiers archivés ici : modèles SQL, scripts de migration initiaux, etc.
